## Pycharm:
> 打开软件进行设置，默认使用utf-8格式
> File->settings->Editor->File and Code Templates->Python Script
```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-
# @Time    : ${DATE} ${TIME}
# @Author  : Aries
# @Site    : ${SITE}
# @File    : ${NAME}.py
# @Software: ${PRODUCT_NAME}
```
